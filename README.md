 &nbsp;&nbsp;如何仅用10台左右的机器，在几秒到几十秒的时间内，分析百亿级别的数据？<br>
 &nbsp;&nbsp;海狗（Higo）是一个分布式的在线分析查询系统，基于hadoop,lucene,solr,蓝鲸等开源系统作为实现，类SQL的查询语法。 海狗是一个能够对大量数据进行分布式处理的软件框架。海狗是快速的高性能的，他的底层使用了索引技术，数据扫描的速度大为增加。Higo是分布式的，它以并行的方式工作，通过并行处理加快处理速度。<br>
